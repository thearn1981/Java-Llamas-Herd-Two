# Farm Manager V3.2

## How to Run

**⚠️ Important: This application must be run in Visual Studio Code with Java 8 or higher**

- **DO NOT** attempt to run in GitHub Codespaces - it will not work properly due to GUI limitations
- Use Visual Studio Code with the Java Extension Pack installed
- Requires Java 8+ (Java 11 or 17 recommended)

## Running the Application

1. Open this folder in Visual Studio Code
2. Ensure you have Java 8+ installed and configured
3. Run `Main.java` to start the Farm Management System

## Features

- Inventory Management
- Services Tracking  
- Animal Management
- Customer Database
- Invoice Generation
